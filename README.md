# 🏛️ Roman Frame Player

一个交互式罗马数字逐帧播放器，支持无限大数、多级上划线表示法、全屏沉浸模式、可调速、可变步长以及自定义字母颜色。

适合用于演示、教学、创意可视化，或单纯享受罗马数字的韵律之美。

## ✨ 功能特性

- **无限大数支持**  
  基于 `BigInt` 实现，可表示任意正整数（仅受浏览器性能限制）。

- **多层次上划线记法 (Vinculum)**  
  自动为千位及以上部分添加上划线，层级超过 5 时切换为“一条横线 + 小型罗马数字”的简化标记，若标记本身 ≥ 4000 则显示竖线。

- **逐帧播放与步长控制**  
  可设定播放速率 (0.5 ~ 60 fps) 和每次变化的步长（任意正整数），实现快速浏览大范围数字。

- **循环与反向播放**  
  支持正向/反向播放，以及到达边界时自动循环。

- **全屏沉浸模式**  
  按 `F` 键或点击按钮进入全屏，纯黑背景仅显示罗马数字，适合大屏演示或氛围营造。

- **颜色自定义**  
  可独立设置每个罗马字母（I, V, X, L, C, D, M）的颜色，并内置彩虹等预设主题。

- **键盘快捷键**  
  空格（播放/暂停）、左右箭头（逐帧进退）、R（重置）、L（循环开关）、F（全屏）。

## 📦 使用方法

1. 克隆仓库或直接下载 `index.html` 文件。
2. 用现代浏览器（Chrome、Firefox、Edge 等）打开 `index.html`。
3. 在设置面板中调整数字范围、速率、步长和颜色。
4. 使用按钮或键盘快捷键控制播放。

无需任何依赖，完全离线可用。

## 🛠️ 技术栈

- 原生 HTML5 / CSS3 / JavaScript (ES2020+)
- `BigInt` 用于超大整数运算
- `Fullscreen API` 实现全屏沉浸
- 纯前端实现，零外部依赖
